## ⚙️ Setting up your system

The software requires a machine running Windows 10 or Windows 11. Make sure you have the latest version of the .NET Desktop Runtime installed. Most computers have this component pre-installed, but if the app fails to open, download the runtime from the official Microsoft website.

You also need at least 50MB of free disk space on your drive. The manager is portable, so you do not need to perform a complex installation process. 

## 🚀 Getting started

Once the file finishes downloading, move it to a folder where you want your mod tools to live. Double-click the file to open the interface. 

The first time you run the tool, it will ask you to point to your Crimson Desert game folder. Click the Browse button and locate the folder where you installed the game. This usually lives in your Steam or Epic Games installation directory. Once selected, click Confirm. 

The software will now scan your game files to ensure it understands your current setup. It will create a backup folder inside the installation path to protect your base game files from accidental changes.

## 📦 Installing your first mod

Download the mod files you want to use. Many mod developers provide these files as compressed folders. Use your file explorer to unzip the contents. Most mod files include a JSON configuration file.

Inside the mod manager, click the Add Mod button. Select the JSON file you downloaded. The manager looks at the file and determines what that mod changes in Crimson Desert. It will check if any existing mods perform similar actions. If it finds a overlap, it will alert you to a potential conflict.

Once added, the mod appears in your library list. Click the checkbox next to the mod name to toggle it on. The tool will apply the necessary changes to your game files. 

## 🔄 Using the mod manager

You can enable or disable any mod at any time. When you disable a mod, the tool restores the original files from the backup it created earlier. This ensures your game returns to its clean state.

The manager also handles load order. If a mod requires another mod to function, the list will help you arrange them in the correct sequence. You can drag and drop entries in the list to change their priority. Mods at the top of the list take priority over mods at the bottom.

## 🛡️ Conflict detection and safety

Modding can sometimes cause game crashes. This software monitors the files you modify to prevent errors. If you install two mods that change the same game data, the conflict detection window pops up. It will describe the conflict in clear terms and suggest which mod to favor. 

If a mod causes issues in the game, you can use the Restore button. This feature overwrites your game files with the original versions saved during the initial setup.

## 💡 Troubleshooting common issues

If the tool does not see your game folder, double-check your path settings. Sometimes game launchers hide the real location. Look for the folder that contains the main game executable file.

If the app freezes, close it using the Task Manager and restart. The software writes a log file whenever it makes a change. You can find this file in the logs folder created alongside your manager. 

If mods appear inactive, ensure you have checked the box next to their name in the main list. You must click the Apply Changes button after checking or unchecking any items for the manager to update the game files.

## 📈 Keeping your mods updated

Check the download source for your specific mods regularly. If a mod developer releases a new version, remove the old version from the manager and add the new one. The manager will detect that you have the same mod and offer to replace the old settings with the updated ones.

## 📝 Frequently asked questions

Does this tool change my game files permanently?
No. The tool creates a backup of your original game files. You can restore them anytime using the interface.

Do I need to be online?
No. You only need an internet connection to download the manager and your mod files. Once the files are on your computer, you can manage them while offline.

Will this get me banned in online games?
Crimson Desert has specific rules regarding mods. Check the game documentation or community forums before using mods in multiplayer modes. This tool is best suited for single-player content.

How many mods can I install?
You can install as many as your computer storage allows. The system uses indexed files to keep the manager running smoothly even with hundreds of mods active.

Can I move the manager to another computer?
Yes. Move the folder to your new computer and point the settings to the new game folder location. Your mod settings should transfer as long as you keep the JSON files together.
